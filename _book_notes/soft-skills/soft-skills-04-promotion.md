## 如何晋升

### 策略一：承担更多责任

> 在任何公司里能让你脱颖而出的最重要法宝就是承担更多的责任。
{: .prompt-tip }

没有人愿意涉足的领域是搜寻机会最好的地方：
- 可能有一个没人愿意碰的遗留应用
- 或者代码库里的某个特别令人讨厌的模块
- 正因为没有人愿意碰，所以这些就成为你日益强大的帝国的领地

**如果你能把沼泽变为良田，你也就展现了自己的价值。**

> **如何让自己承担更多责任**
>
> - 有一个不受重视的项目，你能去负责它吗？
> - 你能帮助团队里的新人快速成长吗？
> - 你能负责文档制作流程，并保证及时更新这些文档吗？
> - 哪项工作是没有人愿意去做，你愿意承担起来，并将其简化或者自动化的？
{: .prompt-tip }

另一种间接承担责任的方式是**成为团队中其他人的导师**，自愿帮助新人加速成长，为任何有需要的人提供帮助。

### 策略二：引人注目

> **提升可见度的方法**
>
> - 每天都记录自己的活动日志——把这个日志以周报的形式发送你的经理
> - 提供演讲或培训——选择一个对你的团队有用的话题
> - 发表意见——只要在会议上就这么做，或者只要你能得到的机会就这么做
> - 保证"曝光度"——定期与老板会面，确保你经常被注意到
{: .prompt-tip }

### 策略三：成为解决方案提供者

在任何组织中，总是有很多人会告诉你：
- 为什么这个想法行不通
- 为什么那个问题太难

**千万不要成为他们中的一员。**

相反，你要成为：
- 那个永远能为各种问题找到解决方案的人
- 勇于执行这些解决方案以获得成果的人

### 策略四：持续学习

> **不断学习**
>
> 目前你正在自己学习些什么？确定要自学的最有价值的东西是什么，制订一份下一年的自学计划。
{: .prompt-danger }

不断增加自己的技能和知识。在你不断提高自己的教育水平时，很难停滞不前。自学能让升职加薪变得容易，因为你可以很清楚地表明：**现在的自己比之前更有价值**。

## 专业素养

> 成为专业人士是一种心态。如果我们总是与恐惧、自毁、拖延和自我怀疑做斗争，那么问题就是：我们正在像外行那样思考问题。外行毫不起眼，外行废话连篇，外行屈从于逆境。专业人士可不这么想。不管怎样，他引人注目，他恪尽职守，他始终如一。
{: .prompt-info }

### 专业人士 vs 外行

| 特点类别     | 专业人士                                 | 外行                                   |
| ------------ | ---------------------------------------- | -------------------------------------- |
| **工作态度** | • 遵守自己的原则<br>• 专注于正确完成工作 | • 让干什么就干什么<br>• 专注于完成工作 |
| **错误处理** | • 不惧怕承认自己错了，不会文过饰非       | • 不懂装懂                             |
| **行为表现** | • 持续稳定                               | • 无法预测，不可靠                     |
| **责任承担** | • 勇于承担责任                           | • 回避责任                             |

### 培养专业习惯

1. **做好计划**：每天提前做好计划，就能养成有效管理时间的习惯。专业人士知道每天必须做什么工作，并且能估算出每项工作大约要花多长时间。

2. **高品质标准**：作为一位专业人士，你必须不断改善和提高自己的工作品质。许多软件开发人员犯的一个巨大错误就是，当无法达到标准时，他们不是完善自己、迎接挑战，而是降低标准。

3. **全面品质管理**：将品质管理应用到你工作的每个细节，而不仅仅是那些看似重要的部分。如果你在某个领域降低了自己的标准，那么你最终会不经意间在其他领域也降低标准。

> 你做的每一件事情就是你所做的一切。
{: .prompt-info }

4. **发挥长处**：专业人士对自己的能力和弱点有着良好、精准而又客观的自我评估。你当然可以改善你的弱点，但最好了解自身的强项是什么并且充分发挥自己的优势。

5. **持续自我完善**：专业人士是通过持续不断的自我完善达到自己所追求的高品质的。确保自己制订并执行了学习计划，拓展自己的技能、学习新东西。不要沾沾自喜，永远对努力成为更好的自己心存渴望。

## 实践清单

### 晋升行动

- [ ] 识别团队中没人愿意做的工作
- [ ] 主动承担导师角色
- [ ] 建立活动日志习惯
- [ ] 寻找演讲或分享的机会
- [ ] 定期与上级沟通工作进展
- [ ] 制定学习计划

### 专业素养培养

- [ ] 每天制定工作计划
- [ ] 为自己的工作设定高标准
- [ ] 坚持学习新技能
- [ ] 客观评估自己的优劣势
- [ ] 保持一致性和可靠性

## 关键收获

1. **承担责任是脱颖而出的关键** - 特别是那些别人不愿意做的工作
2. **可见度很重要** - 让领导知道你在做什么
3. **成为解决问题的人** - 而不是抱怨问题的人
4. **专业素养是一种心态** - 体现在工作的每一个细节
5. **持续改进是专业人士的标志** - 永不满足于现状
